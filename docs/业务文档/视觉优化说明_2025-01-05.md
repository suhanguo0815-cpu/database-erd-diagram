## 核心设计语言

### 1. 字体系统
- **主字体**：从 Inter 升级到 **Outfit**
- **特点**：现代、几何化、可读性强
- **权重**：400/500/600/700/800
- **Fallback**：-apple-system, sans-serif

### 2. 色彩方案

#### 主题色（绿色系）
- `#10b981` - 主绿色（Emerald 500）
- `#059669` - 深绿色（Emerald 600）
- `#f0fdf4` - 浅绿背景（Emerald 50）
- 渐变：`from-emerald-500 to-teal-500`

#### 辅助色
- **橙色**：`#f97316` - 用于警示和次要强调
- **蓝色**：`#3b82f6` - 用于信息展示
- **紫色**：`#a855f7` - 用于特殊功能
- **黄色**：`#fbbf24` - 用于提示和高亮

#### 中性色
- **文字**：`#111827` (Gray-900), `#6b7280` (Gray-500)
- **背景**：`#ffffff` (White), `#f9fafb` (Gray-50)
- **边框**：`rgba(0, 0, 0, 0.04)`

### 3. 圆角系统
- **超大圆角**：`24px` (rounded-3xl) - 主要卡片
- **大圆角**：`20px` (rounded-2xl) - 次要元素
- **中圆角**：`16px` (rounded-xl) - 按钮和小卡片
- **小圆角**：`12px` (rounded-lg) - 标签和徽章

### 4. 阴影系统
- **轻阴影**：`0 2px 16px rgba(0, 0, 0, 0.06)` - 卡片默认
- **中阴影**：`0 4px 24px rgba(0, 0, 0, 0.08)` - 卡片hover
- **重阴影**：`0 10px 40px rgba(0, 0, 0, 0.12)` - 重要元素
- **彩色阴影**：`shadow-emerald-500/30` - 绿色主题元素

### 5. 间距系统
- **页面边距**：`px-6` (24px)
- **卡片内边距**：`p-6` (24px)
- **元素间距**：`space-y-4` (16px) 或 `gap-4`
- **组间距**：`mb-8` (32px)

## 设计原则

### 1. 一致性原则
- 所有页面使用统一的字体系统（Outfit）
- 所有页面使用统一的绿色主题
- 所有卡片使用统一的圆角和阴影

### 2. 层级原则
- 使用字号大小区分信息层级
- 使用颜色深浅区分重要程度
- 使用阴影深度区分交互状态

### 3. 呼吸感原则
- 增大卡片间距，让页面更舒适
- 增大内边距，让内容更易读
- 使用渐变背景，减少视觉疲劳

### 4. 反馈原则
- 卡片添加 hover 效果
- 按钮添加 active 状态
- 进度条使用动画过渡

## 技术实现

### CSS特性使用
- **渐变背景**：`linear-gradient()`
- **过渡动画**：`transition: all 0.2s`
- **transform动效**：`translateY()`, `scale()`
- **毛玻璃效果**：`backdrop-filter: blur()`
- **阴影叠加**：多层 box-shadow

### Tailwind类使用
- **spacing**：p-*, m-*, space-*, gap-*
- **sizing**：w-*, h-*
- **colors**：emerald-*, gray-*, 自定义渐变
- **effects**：shadow-*, rounded-*, opacity-*
- **typography**：text-*, font-*, tracking-*
