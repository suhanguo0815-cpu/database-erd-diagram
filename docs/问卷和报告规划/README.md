## 🎯 核心内容概览

### 健康风险评分体系

**评分类型**: 风险评分（分数越高，风险越高）  
**评分范围**: 每个维度 0-100分  
**评分维度**: 5个风险维度，雷达图展示

| 维度 | 主要评估内容 | 数据来源 |
|------|------------|---------|
| **炎症风险** | 促炎食物摄入、抗炎食物比例、烹饪方式、慢性炎症指标 | 饮食记录分析、问卷数据 |
| **生活习惯风险** | 睡眠质量、作息规律、压力管理、其他生活习惯 | 问卷数据、五因子评估 |
| **营养结构风险** | 宏量营养素失衡、关键营养素缺乏、营养摄入不均衡、饮食模式 | 饮食记录、营养素分析、问卷数据 |
| **BMI风险** | BMI值、腰围、腰臀比、体脂率、体重变化 | 身体测量数据、问卷数据 |
| **代谢风险** | 代谢疾病史、五因子评估-代谢因子、PCOS/甲减风险、代谢相关生活方式 | 问卷数据、五因子评估、体检报告 |

**风险等级划分**（每个维度独立）:
- 0-30分：低风险（绿色）- 继续保持
- 31-60分：中风险（黄色）- 需要关注
- 61-80分：高风险（橙色）- 需要改善
- 81-100分：极高风险（红色）- 急需改善

**注意**: 
- 这是**风险评分**，分数越高表示该维度的健康风险越高
- 五个维度在雷达图上形成五边形展示
- 中心显示综合风险评分（五个维度的平均值）

### 补剂推荐规则

**推荐原则**:
1. 基于数据驱动（问卷、饮食记录、体检报告）
2. 个性化定制（年龄、性别、健康状况）
3. 安全优先（避免过量、考虑药物相互作用）
4. 优先级排序（P0/P1/P2）

**推荐触发条件**:
- 营养素缺乏（维生素D、钙、铁、镁、锌等）
- 健康状况（PCOS、甲减、便秘、失眠等）
- 生活方式（熬夜、运动量大、素食等）

**推荐评分系统**:
- 缺乏程度（0-40分）
- 健康状况相关性（0-30分）
- 饮食摄入情况（0-20分）
- 安全性和必要性（0-10分）

**推荐阈值**:
- ≥60分：强烈推荐（P0）
- 40-59分：建议推荐（P1）
- 20-39分：可选推荐（P2）
- <20分：不推荐
