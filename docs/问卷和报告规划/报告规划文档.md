## 一、报告概述

### 1.1 报告目的
基于用户问卷数据、健康档案、饮食记录等信息，生成个性化的健康评估报告，包含：
- 健康风险评分
- 补剂推荐方案
- 饮食建议
- 运动建议
- 生活方式改善建议

### 1.2 报告类型

| 报告类型 | 触发时机 | 更新频率 | 主要用途 |
|---------|---------|---------|---------|
| **初始评估报告** | 完成问卷后 | 一次性 | 首次健康评估，建立基线 |
| **定期健康报告** | 每周/每月 | 定期 | 跟踪健康变化趋势 |
| **专项分析报告** | 用户主动请求 | 按需 | 针对特定问题的深度分析 |

## 三、健康风险评分规则

### 3.1 评分体系总览

**评分类型**: 风险评分（分数越高，风险越高）  
**评分范围**: 0-100分（每个维度）  
**评分维度**: 5个风险维度，雷达图展示

| 维度 | 说明 | 数据来源 |
|------|------|---------|
| **炎症风险** | 评估饮食中的促炎因素和抗炎能力 | 饮食记录分析、抗炎食物比例 |
| **生活习惯风险** | 评估作息、睡眠、压力等生活习惯 | 问卷数据、睡眠时长、熬夜频率 |
| **营养结构风险** | 评估营养素摄入的均衡性和达标情况 | 营养素摄入分析、宏量营养素比例 |
| **BMI风险** | 评估体重相关的健康风险 | BMI值、腰围、体脂率 |
| **代谢风险** | 评估代谢相关疾病风险 | 问卷数据、五因子评估、代谢疾病史 |

**注意**: 
- 这是**风险评分**，分数越高表示该维度的健康风险越高
- 0-30分：低风险（绿色）
- 31-60分：中风险（黄色）
- 61-80分：高风险（橙色）
- 81-100分：极高风险（红色）

---

### 3.2 炎症风险评分规则（0-100分）

**评分说明**: 分数越高，炎症风险越高

#### 3.2.1 评分维度

**1. 促炎食物摄入频率** (0-40分)

| 促炎食物摄入情况 | 风险分数 | 说明 |
|----------------|---------|------|
| 几乎不摄入 | 0-10分 | 很少或几乎不摄入促炎食物 |
| 偶尔摄入 | 11-25分 | 每周1-2次摄入炸物、高糖饮品等 |
| 经常摄入 | 26-35分 | 每周3-5次摄入促炎食物 |
| 频繁摄入 | 36-40分 | 几乎每天摄入促炎食物 |

**促炎食物定义**:
- 重度促炎：炸物、高糖饮品、加工肉类、反式脂肪、精制糖
- 轻度促炎：精制主食、奶油酱汁、高盐食物

**2. 抗炎食物比例** (0-30分，比例越低风险越高)

| 抗炎食物占比 | 风险分数 | 说明 |
|------------|---------|------|
| ≥50% | 0-10分 | 抗炎食物占主导，炎症风险低 |
| 35-50% | 11-18分 | 抗炎食物较多，炎症风险较低 |
| 20-35% | 19-25分 | 抗炎食物适中，炎症风险中等 |
| 5-20% | 26-30分 | 抗炎食物较少，炎症风险较高 |
| <5% | 30分 | 抗炎食物极少，炎症风险高 |

**抗炎食物定义**:
- 蔬菜：深色绿叶菜、西兰花、番茄、洋葱等
- 优质蛋白：鱼类、瘦肉、豆类
- 健康脂肪：橄榄油、坚果、牛油果
- 全谷物：糙米、燕麦、全麦
- 水果：蓝莓、草莓、樱桃等

**3. 烹饪方式风险** (0-20分)

| 烹饪方式 | 风险分数 | 说明 |
|---------|---------|------|
| 抗炎烹饪为主 | 0-5分 | 清蒸、炖煮、凉拌、水煮、轻炒 |
| 混合烹饪 | 6-12分 | 抗炎与促炎方式混合 |
| 促炎烹饪为主 | 13-20分 | 高温烧烤、重油炒制、油炸比例≥30% |

**4. 慢性炎症指标** (0-10分)

| 指标情况 | 风险分数 | 说明 |
|---------|---------|------|
| 无相关症状 | 0-2分 | 无慢性炎症相关症状 |
| 轻度症状 | 3-5分 | 偶尔有关节不适、疲劳等 |
| 中度症状 | 6-8分 | 经常有炎症相关症状 |
| 重度症状 | 9-10分 | 确诊慢性炎症疾病或频繁症状 |

#### 3.2.2 炎症风险总分计算

```
炎症风险 = 促炎食物摄入频率得分 + 抗炎食物比例得分 + 烹饪方式风险得分 + 慢性炎症指标得分
```

**数据来源**:
- 用户饮食记录（AI识别餐食）
- 每日餐食分析数据（7天/30天平均）
- 问卷数据（慢性疾病、症状等）

### 3.3 生活习惯风险评分规则（0-100分）

**评分说明**: 分数越高，生活习惯风险越高

#### 3.3.1 评分维度

**1. 睡眠质量** (0-30分)

| 睡眠情况 | 风险分数 | 说明 |
|---------|---------|------|
| 睡眠充足且规律 | 0-8分 | 7-9小时，规律作息 |
| 睡眠时间不足或偶尔不规律 | 9-18分 | 6-7小时或偶尔熬夜 |
| 睡眠严重不足或经常不规律 | 19-25分 | <6小时或经常熬夜 |
| 睡眠严重紊乱 | 26-30分 | 严重失眠或作息完全紊乱 |

**评估指标**:
- 睡眠时长（问卷：`sleepDuration`）
- 熬夜频率（问卷：`stayingUpLateFrequency`）
- 睡眠质量自评

**2. 作息规律性** (0-25分)

| 作息情况 | 风险分数 | 说明 |
|---------|---------|------|
| 作息非常规律 | 0-6分 | 每天固定时间睡觉和起床 |
| 作息基本规律 | 7-12分 | 大部分时间规律，偶尔不规律 |
| 作息不规律 | 13-20分 | 经常不规律，时间差异大 |
| 作息完全紊乱 | 21-25分 | 作息时间完全无规律 |

**3. 压力管理** (0-25分)

| 压力情况 | 风险分数 | 说明 |
|---------|---------|------|
| 压力小，管理良好 | 0-8分 | 压力小，有良好的应对方式 |
| 压力中等，基本可控 | 9-15分 | 有一定压力，基本能应对 |
| 压力较大，难以应对 | 16-22分 | 压力大，经常感到焦虑 |
| 压力极大，严重影响 | 23-25分 | 压力极大，严重影响生活 |

**评估指标**:
- 五因子评估中的情绪因子
- 主观状态（问卷：`selfPerceivedHealthStatus`）
- 头晕乏力频率（问卷：`dizzinessFatigueFrequency`）

**4. 其他生活习惯** (0-20分)

| 习惯情况 | 风险分数 | 说明 |
|---------|---------|------|
| 生活习惯良好 | 0-5分 | 无不良习惯 |
| 有1-2项不良习惯 | 6-12分 | 如久坐、缺乏日晒等 |
| 有多项不良习惯 | 13-18分 | 多项不良习惯叠加 |
| 生活习惯极差 | 19-20分 | 多项严重不良习惯 |

**评估指标**:
- 日晒时间（问卷：`sunExposureMinutes`）
- 职业活动水平（问卷：`occupationalActivityLevel`）
- 排便情况（问卷：`bowelHabits`）

#### 3.3.2 生活习惯风险总分计算

```
生活习惯风险 = 睡眠质量得分 + 作息规律性得分 + 压力管理得分 + 其他生活习惯得分
```

**数据来源**:
- 问卷数据（睡眠、作息、压力相关）
- 五因子评估数据
- 用户行为数据

### 3.4 营养结构风险评分规则（0-100分）

**评分说明**: 分数越高，营养结构风险越高

#### 3.4.1 评分维度

**1. 宏量营养素失衡** (0-35分)

| 失衡情况 | 风险分数 | 说明 |
|---------|---------|------|
| 三大营养素均衡 | 0-8分 | 蛋白质、碳水、脂肪比例合理 |
| 轻度失衡 | 9-18分 | 某一类营养素偏高或偏低 |
| 中度失衡 | 19-28分 | 明显偏重某一类营养素 |
| 严重失衡 | 29-35分 | 营养结构严重失衡 |

**评估标准**:
- 蛋白质：推荐量±10%为正常
- 碳水化合物：推荐量±15%为正常
- 脂肪：推荐量±15%，且健康脂肪占比≥60%

**2. 关键营养素缺乏** (0-30分)

| 缺乏情况 | 风险分数 | 说明 |
|---------|---------|------|
| 营养素充足 | 0-8分 | 关键营养素均达标 |
| 1-2种营养素轻度缺乏 | 9-15分 | 轻度缺乏，影响较小 |
| 3-4种营养素缺乏 | 16-24分 | 多种营养素缺乏 |
| 多种营养素严重缺乏 | 25-30分 | 严重缺乏，健康风险高 |

**关键营养素**:
- 矿物质：镁、锌、钾、钠、钙、铁
- 维生素：维生素D、维生素B12、叶酸、维生素C

**3. 营养素摄入不均衡** (0-20分)

| 均衡情况 | 风险分数 | 说明 |
|---------|---------|------|
| 营养均衡 | 0-5分 | 各类食物摄入均衡 |
| 轻度不均衡 | 6-11分 | 某些食物类别摄入不足 |
| 中度不均衡 | 12-17分 | 明显偏食或挑食 |
| 严重不均衡 | 18-20分 | 严重偏食，营养单一 |

**评估指标**:
- 全谷占比（问卷：`wholeGrainRatio`）
- 豆类频率（问卷：`legumeFrequency`）
- 蔬菜水果摄入（问卷：`fruitVegetableIntake`）
- 奶制品摄入（问卷：`dairyIntakeMl`）

**4. 饮食模式风险** (0-15分)

| 饮食模式 | 风险分数 | 说明 |
|---------|---------|------|
| 健康饮食模式 | 0-3分 | 规律三餐，营养均衡 |
| 一般饮食模式 | 4-8分 | 基本规律，偶尔不规律 |
| 不健康饮食模式 | 9-12分 | 经常不规律，饮食质量差 |
| 极不健康饮食模式 | 13-15分 | 饮食完全无规律，质量极差 |

**评估指标**:
- 饮食模式（问卷：`dietPattern`）
- 餐食安排（问卷：`mealArrangement`）
- 饮食习惯（问卷：`dietaryHabits`）

#### 3.4.2 营养结构风险总分计算

```
营养结构风险 = 宏量营养素失衡得分 + 关键营养素缺乏得分 + 营养素摄入不均衡得分 + 饮食模式风险得分
```

**数据来源**:
- 饮食记录分析
- 营养素摄入数据
- 问卷数据（饮食习惯相关）

### 3.5 BMI风险评分规则（0-100分）

**评分说明**: 分数越高，BMI相关健康风险越高

#### 3.5.1 BMI计算

```
BMI = 体重(kg) / 身高(m)²
```

#### 3.5.2 评分标准

| BMI范围 | 风险等级 | 风险分数 | 说明 |
|---------|---------|---------|------|
| 18.5-23.9 | 正常 | 0-20分 | 健康体重范围，风险最低 |
| 24.0-27.9 | 超重 | 21-45分 | 轻度超重，需注意 |
| 17.0-18.4 | 偏瘦 | 15-35分 | 体重偏轻，可能有营养不良风险 |
| 28.0-31.9 | 肥胖I级 | 46-70分 | 中度肥胖，健康风险较高 |
| <17.0 | 严重偏瘦 | 36-60分 | 营养不良风险高 |
| ≥32.0 | 肥胖II级+ | 71-100分 | 重度肥胖，健康风险极高 |

#### 3.5.3 额外风险因素加分

| 风险因素 | 加分 | 说明 |
|---------|------|------|
| 腰围超标（男性>90cm，女性>85cm） | +10分 | 中心性肥胖风险 |
| 腰臀比异常（男性>0.9，女性>0.85） | +10分 | 腹部脂肪堆积风险 |
| 体脂率异常 | +5-15分 | 根据体脂率异常程度 |
| 近期体重快速变化（>5kg/月） | +10分 | 体重波动风险 |

#### 3.5.4 特殊人群调整

**女性特殊考虑**:
- 孕期/哺乳期：BMI风险评分权重降低
- PCOS患者：BMI风险评分权重提高（代谢风险更高）

**数据来源**:
- 当前体重（问卷：`initialWeightKg`）
- 身高（问卷：`heightCm`）
- 腰围（问卷：`waistCircumferenceCm`）
- 臀围（问卷：`hipCircumferenceCm`）

### 3.6 代谢风险评分规则（0-100分）

**评分说明**: 分数越高，代谢相关疾病风险越高

#### 3.6.1 评分维度

**1. 代谢疾病史** (0-30分)

| 疾病情况 | 风险分数 | 说明 |
|---------|---------|------|
| 无代谢疾病 | 0-5分 | 无相关疾病史 |
| 有代谢疾病家族史 | 6-12分 | 家族中有相关疾病 |
| 有轻度代谢异常 | 13-20分 | 如血糖偏高、血脂异常等 |
| 已确诊代谢疾病 | 21-30分 | 如糖尿病、高血压等 |

**评估指标**:
- 确诊疾病（问卷：`healthConstraints.diagnosed`）
- 慢性疾病（问卷：`chronicDiseases`）
- 体检报告数据

**2. 五因子评估-代谢因子** (0-25分)

| 代谢因子得分 | 风险分数 | 说明 |
|------------|---------|------|
| 代谢因子良好（16-20分） | 0-8分 | 代谢功能良好 |
| 代谢因子一般（12-15分） | 9-15分 | 代谢功能一般 |
| 代谢因子较差（8-11分） | 16-22分 | 代谢功能较差 |
| 代谢因子差（4-7分） | 23-25分 | 代谢功能差，风险高 |

**数据来源**: 五因子评估中的代谢因子得分

**3. PCOS/甲减相关风险** (0-25分，仅相关人群)

| 情况 | 风险分数 | 说明 |
|------|---------|------|
| 无相关症状 | 0-5分 | 无PCOS或甲减相关症状 |
| 有轻度症状 | 6-12分 | 有1-2项相关症状 |
| 有中度症状 | 13-20分 | 有3-4项相关症状 |
| 已确诊或症状严重 | 21-25分 | 已确诊或症状严重 |

**评估指标**:
- PCOS症状（问卷：`healthConstraints.pcos`）
- 甲减症状（问卷：`healthConstraints.hypot`）
- 库欣综合征症状（问卷：`healthConstraints.cushing`）

**4. 代谢相关生活方式** (0-20分)

| 生活方式 | 风险分数 | 说明 |
|---------|---------|------|
| 生活方式健康 | 0-5分 | 运动充足，饮食健康 |
| 生活方式一般 | 6-12分 | 运动不足或饮食不健康 |
| 生活方式不健康 | 13-18分 | 久坐、缺乏运动、饮食差 |
| 生活方式极不健康 | 19-20分 | 多项不健康因素叠加 |

**评估指标**:
- 运动频率（问卷：`exerciseDaysPerWeek`）
- 运动时长（问卷：`exerciseDurationMinutes`）
- 职业活动水平（问卷：`occupationalActivityLevel`）
- 饮食习惯（问卷：`dietPattern`、`tastePreference`等）

#### 3.6.2 代谢风险总分计算

```
代谢风险 = 代谢疾病史得分 + 五因子评估-代谢因子得分 + PCOS/甲减相关风险得分 + 代谢相关生活方式得分
```

**数据来源**:
- 问卷数据（代谢疾病相关）
- 五因子评估数据
- 体检报告数据
- 生活方式问卷数据

### 3.7 健康风险评分等级划分

**每个维度独立评分**（0-100分）：

| 风险分数范围 | 风险等级 | 颜色 | 风险提示 |
|------------|---------|------|---------|
| 0-30分 | 低风险 | 绿色 #6FC86B | 继续保持，风险较低 |
| 31-60分 | 中风险 | 黄色 #eab308 | 需要关注，适当改善 |
| 61-80分 | 高风险 | 橙色 #f97316 | 需要改善，风险较高 |
| 81-100分 | 极高风险 | 红色 #ef4444 | 急需改善，风险极高 |

**雷达图展示**:
- 五个维度在雷达图上形成五边形
- 每个维度得分对应雷达图的一个顶点
- 中心显示综合风险评分（五个维度的平均值或加权平均）

## 四、补剂推荐规则

### 4.1 补剂推荐原则

1. **基于数据驱动**: 根据问卷数据、饮食记录、体检报告等客观数据推荐
2. **个性化定制**: 考虑用户年龄、性别、健康状况、饮食偏好等
3. **安全优先**: 避免过量补充，考虑药物相互作用
4. **优先级排序**: 按重要性和紧迫性排序推荐

---

### 4.2 补剂推荐触发条件

#### 4.2.1 营养素缺乏触发

| 营养素 | 缺乏标准 | 推荐补剂 | 优先级 |

| 铁 | 女性月经期 或 血清铁蛋白<15ng/ml | 铁剂 | P0 |
| 镁 | 饮食中镁摄入<推荐量70% | 镁片 | P1 |
| 锌 | 饮食中锌摄入<推荐量70% | 锌片 | P1 |
| 维生素B12 | 素食者 或 血清B12<200pg/ml | 维生素B12 | P0 |
| 叶酸 | 备孕/孕期 或 血清叶酸<4ng/ml | 叶酸 | P0 |
| Omega-3 | 深海鱼<1次/周 | 鱼油/藻油 | P1 |

#### 4.2.2 健康状况触发

| 失眠 | 镁、褪黑素 | 改善睡眠质量 | P1 |
| 疲劳乏力 | 维生素B族、铁、维生素D | 改善能量代谢 | P1 |
| 口腔溃疡 | 维生素B2、B12、锌 | 促进黏膜修复 | P2 |

#### 4.2.3 生活方式触发

### 4.3 补剂推荐算法

#### 4.3.1 评分系统

每个补剂根据以下维度评分（0-100分）：

**1. 缺乏程度** (0-40分)
- 严重缺乏：40分
- 中度缺乏：25分
- 轻度缺乏：10分
- 无缺乏：0分

**2. 健康状况相关性** (0-30分)
- 直接相关：30分
- 间接相关：15分
- 无关：0分

**3. 饮食摄入情况** (0-20分)
- 饮食无法满足：20分
- 饮食部分满足：10分
- 饮食可满足：0分

**4. 安全性和必要性** (0-10分)
- 必需且安全：10分
- 可选且安全：5分
- 需谨慎：0分

#### 4.3.2 推荐排序

```
补剂推荐分数 = 缺乏程度得分 + 健康状况相关性得分 + 饮食摄入情况得分 + 安全性和必要性得分
```

**推荐规则**:
- 分数 ≥ 60分：强烈推荐（P0）
- 分数 40-59分：建议推荐（P1）
- 分数 20-39分：可选推荐（P2）
- 分数 < 20分：不推荐

- 运动频率 → 蛋白质、BCAA推荐

### 4.5 补剂服用建议

#### 4.5.1 服用时间建议

| 补剂类型 | 最佳服用时间 | 说明 |
|---------|------------|------|
| 脂溶性维生素（A、D、E、K） | 随餐服用 | 需要脂肪帮助吸收 |
| 水溶性维生素（B族、C） | 餐后服用 | 减少胃部不适 |
| 矿物质（钙、镁、铁） | 分次服用 | 避免一次性大量摄入 |
| 益生菌 | 餐前30分钟 | 空腹效果更好 |
| 鱼油 | 随餐服用 | 减少鱼腥味 |

#### 4.5.2 相互作用提醒

**需要分开服用的组合**:
- 钙 + 铁：间隔2小时
- 锌 + 铜：间隔2小时
- 维生素C + 维生素B12：间隔1小时

**需要一起服用的组合**:
- 维生素D + 钙：协同作用
- 维生素C + 铁：促进铁吸收

## 五、报告生成流程

### 5.1 数据收集阶段

1. **问卷数据**: 用户完成问卷后的所有数据
2. **饮食记录**: 最近7-30天的饮食记录（如有）
3. **体检报告**: 上传的体检报告数据（如有）
4. **历史数据**: 用户历史健康档案数据

### 5.2 数据分析阶段

1. **健康风险评分计算**: 根据评分规则计算各维度得分
2. **营养素缺乏分析**: 分析各营养素摄入情况
3. **健康状况评估**: 结合问卷和体检数据评估健康状况
4. **补剂需求分析**: 根据缺乏情况和健康状况分析补剂需求

### 5.3 报告生成阶段

1. **生成报告结构**: 根据数据完整性生成报告模块
2. **填充报告内容**: 填充评分、推荐、建议等内容
3. **个性化调整**: 根据用户特征调整报告语言和重点
4. **格式化输出**: 生成HTML/PDF格式报告

### 5.4 报告展示阶段

1. **报告首页**: 展示健康评分总览
2. **详情页面**: 各模块详情页面
3. **交互功能**: 支持分享、下载、打印等
