# 数据库表关系图 - 发布指南

## 方法一：GitHub Pages（推荐）

### 步骤：

1. **在GitHub上创建新仓库**
   - 访问 https://github.com/new
   - 仓库名：`database-erd-diagram` （或您喜欢的名称）
   - 选择 Public（公开）
   - 不要初始化 README、.gitignore 或 license

2. **推送代码到GitHub**
   ```bash
   cd "/Users/guotongxue/Desktop/AI营养师小程序"
   git remote add origin https://github.com/您的用户名/database-erd-diagram.git
   git branch -M main
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 在GitHub仓库页面，点击 Settings（设置）
   - 左侧菜单找到 Pages
   - Source（源）选择 "Deploy from a branch"
   - Branch（分支）选择 `main`，文件夹选择 `/ (root)`
   - 点击 Save（保存）

4. **访问您的网站**
   - 几分钟后，您的网站将发布在：
   - `https://您的用户名.github.io/database-erd-diagram/`
   - 或直接访问：`https://您的用户名.github.io/database-erd-diagram/数据库表关系图.html`

## 方法二：Netlify（最简单）

1. 访问 https://www.netlify.com/
2. 注册/登录账户
3. 将整个文件夹拖拽到 Netlify 的部署区域
4. 网站会自动发布，获得一个网址（如：`your-site.netlify.app`）

## 方法三：Vercel

1. 访问 https://vercel.com/
2. 注册/登录账户
3. 点击 "New Project"
4. 导入您的GitHub仓库，或直接上传文件夹
5. 自动部署完成

## 方法四：本地预览

直接在浏览器中打开 `数据库表关系图.html` 文件即可查看。

## 文件说明

- `数据库表关系图.html` - 主文件，包含完整的Mermaid图表
- `index.html` - 自动跳转到主文件

## 注意事项

- 确保网络连接正常，因为图表使用CDN加载Mermaid.js
- 如果图表无法显示，请检查浏览器控制台是否有错误
- GitHub Pages可能需要几分钟才能生效
